дёҖгҖҒеүҚиЁҖ

иҝҷйҮҢдёҚеҗҢдәҺе…¶д»–дәәзҡ„GoиҜӯиЁҖе…Ҙй—ЁпјҢзҺҜеўғе®үиЈ…жҲ‘еҗ‘жқҘжіЁйҮҚй…ҚзҪ®пјҢжҜ”еҰӮдҫқиө–еҢ…гҖҒзј“еӯҳзҡ„й»ҳи®Өзӣ®еҪ•гҖӮеӣ дёәеүҚжңҹдёҚеј„еҘҪпјҢеҗҺйқўиҰҒж•ҙзҗҶеҸҲеҪұе“ҚиҝҷеҪұе“ҚйӮЈзҡ„пјҢжүҖд»Ҙе°ұе№Іи„ҶеҶҷжҲҗж–Үз« пјҢж–№дҫҝеҗҺжңҹжҚЎиө·гҖӮ

дәҢгҖҒе®үиЈ…

1.е®үиЈ…еҢ…

2.ж“ҚдҪң

е°Ҷmsiе®үиЈ…еҲ°жҢҮе®ҡдҪҚзҪ®пјҢ D:\Go\binж·»еҠ еҲ°зҺҜеўғеҸҳйҮҸpath

3.йӘҢиҜҒ

cmdиҫ“е…Ҙgo versionжҹҘзңӢзүҲжң¬еҸ·пјҢgo run test.goжү§иЎҢи„ҡжң¬

дёүгҖҒзҺҜеўғй…ҚзҪ®

1.жҹҘзңӢGoеҸҳйҮҸ

2.дҝ®ж”№зҺҜеўғй…ҚзҪ®

йҖҡиҝҮдёҠйқўе‘Ҫд»ӨжҹҘзңӢеҹәжң¬й»ҳи®Өй…ҚзҪ®еҗҺпјҢиҝҷйҮҢе°ұжҳҜз”ЁжқҘдҝ®ж”№goиҝҗиЎҢзҡ„зј“еӯҳдҪҚзҪ®пјҢжЁЎеқ—е®үиЈ…дҪҚзҪ®гҖӮ

еӣӣгҖҒйЎ№зӣ®еҲӣе»ә

1.еҹәжң¬зӣ®еҪ•

2.еҲқе§ӢеҢ–GoжЁЎеқ—

еңЁйЎ№зӣ®зӣ®еҪ•дёӯиҝҗиЎҢ go mod init е‘Ҫд»ӨжқҘеҲқе§ӢеҢ–дёҖдёӘж–°зҡ„ Go жЁЎеқ—гҖӮдҪ йңҖиҰҒжҸҗдҫӣдёҖдёӘе”ҜдёҖзҡ„жЁЎеқ—и·Ҝеҫ„пјҢйҖҡеёёжҳҜд»Јз Ғд»“еә“зҡ„ URLгҖӮ

з”ҹжҲҗдёҖдёӘ go.mod ж–Үд»¶пјҢз”ЁдәҺи®°еҪ•йЎ№зӣ®зҡ„дҫқиө–е…ізі», еҲқе§ӢеҢ–ж—¶еҸӘжңүдёҖдёӘgoзүҲжң¬пјҢеҪ“go getдёӢиҪҪдәҶдҫқиө–еҢ…еҗҺпјҢдјҡиҮӘеҠЁеңЁgo.modж–Үд»¶дёӯи®°еҪ•дҫқиө–еҢ…зҡ„зүҲжң¬еҸ·пјҢе№¶з”ҹжҲҗ.sumж–Үд»¶гҖӮ

3.ж·»еҠ дҫқиө–

еҰӮжһңдҪ зҡ„йЎ№зӣ®йңҖиҰҒдҪҝз”Ёз¬¬дёүж–№еә“пјҢеҸҜд»ҘдҪҝз”Ё go get е‘Ҫд»ӨжқҘж·»еҠ дҫқиө–гҖӮдҫӢеҰӮпјҢеҰӮжһңдҪ жғідҪҝз”Ё github.com/gin-gonic/gin иҝҷдёӘеә“пјҢеҸҜд»ҘиҝҗиЎҢпјҡ

иҝҷе°ҶиҮӘеҠЁжӣҙж–° go.mod ж–Үд»¶пјҢж·»еҠ зӣёеә”зҡ„дҫқиө–йЎ№гҖӮ

4.зј–еҶҷGoзЁӢеәҸ

ж №зӣ®еҪ•дёӢеҲӣе»әmain.go ж–Үд»¶е№¶зј–еҶҷдҪ зҡ„ Go зЁӢеәҸгҖӮд»ҘдёӢжҳҜдёҖдёӘз®ҖеҚ•зҡ„зӨәдҫӢд»Јз ҒпјҢе®ғе°Ҷиҫ“еҮәвҖңHello, World!вҖқпјҡ

5.иҝҗиЎҢGoзЁӢеәҸ

еңЁйЎ№зӣ®зӣ®еҪ•дёӯпјҢдҪҝз”Ё go run е‘Ҫд»ӨжқҘзј–иҜ‘е№¶иҝҗиЎҢдҪ зҡ„ Go зЁӢеәҸпјҡ

6.жһ„е»әGoзЁӢеәҸ (жү“еҢ…exe)

(1). иҝӣе…ҘеҪ“еүҚзӣ®еҪ•

(2). иҫ“е…Ҙе‘Ҫд»Ө go run main.go

(3). дёҚжү“ејҖGUIй»‘жЎҶзҡ„жү“еҢ…е‘Ҫд»Ө

(4). зӣҙжҺҘзј–иҜ‘жү“еҢ…иҝҗиЎҢпјҢжңүй»‘жЎҶ

дә”гҖҒйҷ„еҠ 

1.еҚ•е…ғжөӢиҜ•

Go жҸҗдҫӣдәҶз®ҖеҚ•зҡ„еҚ•е…ғжөӢиҜ•жЎҶжһ¶гҖӮдҪ еҸҜд»ҘеңЁйЎ№зӣ®дёӯж·»еҠ жөӢиҜ•ж–Үд»¶пјҢж–Үд»¶еҗҚйҖҡеёёд»Ҙ _test.go з»“е°ҫгҖӮдҫӢеҰӮпјҢеҲӣе»әдёҖдёӘ main_test.go ж–Үд»¶пјҡ

2.ж јејҸеҢ–д»Јз Ғ

Go жҸҗдҫӣдәҶ gofmt е‘Ҫд»ӨжқҘж јејҸеҢ–д»Јз ҒпјҢдҪҝе…¶з¬ҰеҗҲ Go зҡ„д»Јз ҒйЈҺж ји§„иҢғпјҡ

иҝҷе°ҶиҮӘеҠЁж јејҸеҢ–йЎ№зӣ®зӣ®еҪ•дёӢзҡ„жүҖжңү Go ж–Үд»¶гҖӮ

3.жү“еҢ…зЁӢеәҸдёҚйҖҖеҮә

зј–еҶҷзҡ„GoзЁӢеәҸпјҢдёәдәҶи®©е…¶жү“еҢ…еҗҺexeеә”з”ЁеҗҜеҠЁеҗҺдёҚйҖҖеҮәпјҢеҸҜд»Ҙиҝҷж ·еӨ„зҗҶпјҡ
