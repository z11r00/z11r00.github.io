еүҚиЁҖ

иҷҪз„¶жҲ‘жҳҜеҒҡеҗҺз«Ҝзҡ„пјҢдҪҶд№ҹж—¶еёёе…іжіЁеүҚз«ҜпјҢеҸӘжҳҜжңҖиҝ‘и§үеҫ—еүҚз«Ҝзҡ„еҗ„з§Қе·Ҙе…·еҗҚз§°еӨӘзңјиҠұзјӯд№ұдәҶпјҢnvmгҖҒyarnгҖҒpnpmгҖҒtaroвҖҰвҖҰ

дёҚиҰҒиҜҙе…·дҪ“дҪҝз”ЁпјҢе°ұиҝһд»–д»¬еҲ°еә•жҳҜе·Ҙе…·иҝҳжҳҜж–°жЎҶжһ¶жҲ‘йғҪжҗһж··гҖӮжүҖд»ҘпјҢжҲ‘е°ұз”ЁдёҖзҜҮж–Үз« з®ҖиҰҒи®°еҪ•д»–д»¬еҲ°еә•жҳҜд»Җд№ҲпјҢжҖҺд№ҲеҢәеҲ«д»ҘеҸҠз®Җжҳ“зҡ„жҗӯе»әиҝҮзЁӢгҖӮ

nvm

nvmпјҲNode Version ManagerпјүпјҢйЎҫеҗҚжҖқд№үпјҢд»–жҳҜNode.jsзҡ„зүҲжң¬з®ЎзҗҶе·Ҙе…·гҖӮдё»иҰҒз”ЁжқҘеңЁеҚ•дёӘж“ҚдҪңзі»з»ҹдёӢе®үиЈ…жҲ–з®ЎзҗҶеӨҡзүҲжң¬NodeпјҢеёёз”ЁдәҺи§ЈеҶійЎ№зӣ®дёӯдёҚеҗҢзүҲжң¬дҫқиө–й—®йўҳгҖӮ

дёӢиҪҪ

githubжҗңдҪңиҖ…coreybutlerпјҢйЎ№зӣ®nvm-windowsгҖӮ

е®үиЈ…

е®үиЈ…еүҚеҰӮжһңз”өи„‘е·Із»ҸжңүдәҶNode.jsпјҢжңҖеҘҪеҚёиҪҪд»ҘйҳІеҗҺйқўеҶІзӘҒгҖӮз„¶еҗҺзӮ№еҮ»exeж–Үд»¶дёӢдёҖжӯҘдёӢдёҖжӯҘе®ҢжҲҗеҗҺпјҢnvmдјҡиҮӘеҠЁеңЁз”өи„‘зҡ„з”ЁжҲ·еҸҳйҮҸдёӯж·»еҠ NVM_HOMEе’ҢNVM_SYMLINKпјҢжңҖеҗҺcmdиҫ“е…Ҙnvm -vпјҢжҳҫзӨәnvmзүҲжң¬еҸ·иЎЁзӨәе·Із»ҸжҲҗеҠҹгҖӮ

дҪҝз”Ё

pnpm

pnpmпјҲPerformant npmпјүпјҢжҳҜдёҖдёӘеҝ«йҖҹгҖҒзЈҒзӣҳз©әй—ҙеҸӢеҘҪзҡ„еҢ…з®ЎзҗҶе·Ҙе…·пјҢе’Ңnpmзұ»дјјпјҢеҸҜд»Ҙз®ҖеҚ•зҗҶи§Јдёәnpmзҡ„дјҳеҢ–еҠ ејәзүҲпјҢйҖӮеҗҲдәҺиҫғеӨ§зҡ„еүҚз«ҜйЎ№зӣ®пјҢе…·дҪ“зҡ„зү№еҫҒеҸҜд»ҘзңӢе…¶д»–еҚҡдё»зҡ„д»Ӣз»ҚгҖӮеӣ дёәnpmжҳҜNodeиҮӘеёҰпјҢжүҖд»ҘжңүNodeеҚіеҸҜжңүpnpmгҖӮ

дёӢиҪҪе®үиЈ…

npm install pnpm -g

дҪҝз”Ё

yarn

yarn жҳҜ Facebook ејҖеҸ‘зҡ„дёҖдёӘ JavaScript еҢ…з®ЎзҗҶе·Ҙе…·пјҢyarn дҪҝз”ЁдёҺ npm дёҚеҗҢзҡ„й”Ғе®ҡж–Үд»¶ж јејҸпјҢдҪҶжҳҜд»Қз„¶еҸҜд»Ҙе…је®№ npm зҡ„еҢ…з®ЎзҗҶи§„иҢғгҖӮ

дёӢиҪҪе®үиЈ…

npm install -g yarn

дҪҝз”Ё
